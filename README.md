# NoteBookApp
A notepad app,  this app is the basis of database design of an offline app and all CRUD (create, read, update, delete) operations are available, app is written in kotlin, used are;  
-RecyclerView, <br>
-PopupMenu, <br>
-BottomMenu,  <br>
-SqliteOpenHelper<br>
-MVVM design pattern<br>
